    Java 技术¶

    1. Java 基础 a. 对面向对象编程有深入理解 b. 熟练掌握 Java 集合、IO、异常、反射、自定义注解等 c. 熟悉 常见集合原理。

    2. Java 多线程：并发编程 a. 熟悉 synchronized 原理 volatile 原理、ThreadLocal 原理、Atomic 原子类、AQS、CAS 锁原理 b. 熟悉 常用并发类库 Semaphore / CountDownLatch / CyclicBarrie 原理。 c. 熟悉 线程池的使用和参数调整。

    3. JVM 核心知识 a. 熟悉 Java内存区域、类加载器、字节码、虚拟机垃圾收集算法、虚拟垃圾收集器 b. 实际处理 生产环境下排查 CPU 飙高、内存泄漏问题。
    4. JavaWeb 开发 a. 熟悉 Tomcat、Servlet、Cookies、Session、Filter、JSP 知识 b. 熟悉 Ajax、JSON、XML 等常见 Web 相关技术 流行框架 / 中间件
    5. Spring 生态系框架 a. 熟练 Spring、SpringMVC、SpringBoot 开发，并对其核心原理 / 源码有一定的了解 b. 了解/熟悉 SpringCloud / Alibaba 技术栈（使用经验，但非目前博主所属公司主力技术栈） c. 了解 SpringSecurity、Shiro 等认证授权框架（使用经验）
    6. RPC 框架 a. 熟悉 Dubbo 的语法、配置、调用方式。

    7. ORM 框架 a. 熟练 MyBatis、MybatisPlus b. 了解 JPA

    8. 消息中间件 a. 熟悉 RabbitMQ、RocketMQ 的语法和使用，具有中大型项目使用和配置经验 b. 了解 Kafka 的基本概念和使用

    9. 缓存中间件 a. 熟悉 Redis 语法和使用了，具有大量使用经验
    10. 搜索中间件 a. 了解 ELasticSearch 语法，具有使用经验。 数据库
    11. MySQL a. 熟悉 关系型数据库 MySQL 数据库开发，能够熟练编写常用 SQL 语句 b. 熟悉 MySQL 常见优化手段（索引、SQL优化、分库分表） c. 熟悉 MySQL 核心原理（架构、存储、引擎等）
    12. Redis a. 熟悉 Redis 语法、各种命令，以及与 Java / SpringBoot 的集成 b. 熟悉 各种缓存高并发的使用场景（缓存雪崩，缓存穿透，缓存失效等） c. 了解其配置文件、持久化、发布订阅、主从问题、哨兵等 d. 了解分布式锁，具有 Lua 脚本 + Redis 的开发经验 其它
    13. 设计与开发 a. 具有丰富的系统分析、功能设计能力 b. 代码风格规范、对代码、注释、命名、结构具有严格的自我要求 c. 熟悉常用的设计模式（单例、工厂、建造者、观察者、策略、代理等。）
